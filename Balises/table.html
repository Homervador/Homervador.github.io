Pour cree un tableau il faut utiliser la balise <table> </table> Dans le tableau prealablement cree il faut creer des lignes. L'en-tete et le pied du tableau n'etant pas obligatoire il est conseille de les utiliser. Pour cree l'en-tete on utilise la balise <thead> </thead>. Pour cree une ligne dans l' en-tete on utilise la balise <th> </th>. Dans ces lignes on cree des cellules a l'aide de la balise <td> </td>. Pour cree un pied de tableau on utilise la balise <tfoot> </tfoot>. Pour cree une ligne dans le pied de tableau on utilise la balise <td> </td>. Dans ces ligne on cree des cellules a l'aide de la balise <td> </td>. Pour cree le corp du tableau on utilis la balise <tbody> </tbody>. Pour cree une ligne dans le pied de tableau on utilise la balise <td> </td>. Dans ces ligne on cree des cellules a l'aide de la balise <td> </td>.
